# Passive Information Gathering

## Recon

Risoluzione host da Linux:
``` bash
$ host roccosicilia.com
roccosicilia.com has address 192.0.78.24
roccosicilia.com has address 192.0.78.25
```

Risoluzione host da Windows:
``` powershell
> nslookup roccosicilia.com 8.8.8.8
Server: dns.google
Address: 8.8.8.8

Non-authoritative answer:
Name: roccosicilia.com
Address: 192.0.78.24
  192.0.78.25
```

Analisi file robots.txt, script in python e bash:
 - https://github.com/roccosicilia/rootstash/blob/main/scripts/robots_explore.py
 - https://github.com/roccosicilia/rootstash/blob/main/scripts/robots_explore.sh

Tool whatweb per identificare tecnologie utilizzate dai siti web:
``` bash
$ whatweb roccosicilia.com
```

Whois Enum:
``` bash
$ whois roccosicilia.com
```

DNS recon
``` bash
$ dig roccosicilia.com any
$ dig roccosicilia.com mx
$ dig roccosicilia.com txt
```

WAF recon
``` bash 
$ wafw00f roccosicilia.com
$ nmap --script=http-waf-fingerprint -p443 roccosicilia.com
```

Subdomain enumeration
``` bash
$ sublist3r -d roccosicilia.com
```
